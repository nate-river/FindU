## 手机端功能要求

### 产品定位  

* 让所有人能快速联系到其他人

### 产品功能要点

* 寻找快 侧栏字母导航 模糊搜索及时反馈
* 易用 直接打出电话 自己有小号 优先拨打对方小号 自己没小号 拨打对方电话
* 下拉刷新 与服务器同步
* 离线优先 使用本地存储 存储所有联系人信息 本人信息

### 采用技术


* 布局使用viewprot定死固定尺寸
* js采用jquery2.0 + 百度Touch.js
* 服务器一套ajax接口 使用bom跳转页面
* pc端上服务器  移动端进hbuilder



## PC端功能要求

* 只有管理员可登录，编辑
* 便捷的增删改查
